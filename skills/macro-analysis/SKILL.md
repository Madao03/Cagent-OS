# 宏观分析（Macro Analysis）

> **核心原则：时间周期决定指标权重。** 不同时间周期的宏观分析，参考的重点数据和信息所占权重完全不同。
> 短期（<3月）市场定价指标占主导；中期（3月-2年）景气循环+政策占主导；长期（>2年）结构性因素占主导。

## 何时触发

当用户询问或需要判断以下内容时，自动加载此 Skill：

- 宏观环境当前处于什么阶段（扩张/放缓/滞胀/衰退）
- 地缘政治事件对市场的影响路径
- 外围市场价格联动信号
- 宏观因素对 crypto 或风险资产的间接影响分析
- 美联储人事变动（主席换届、FOMC票委轮换、候选人立场分析）
- 央行政策拐点判断（降息/加息周期、QT/QE转向）
- 用户同时提出多个相互关联的宏观/市场问题
- **⚠️ 隐式触发**：任何涉及"当前宏观环境"的判断都必须先确定时间周期再分析

## 步骤 0：确定时间周期（必须先做）

> 在收集任何数据之前，先判断用户关心的是一段什么样的时间。不同周期用不同的分析框架。

| 时间周期 | 核心驱动力 | 主导指标类型 |
|:-----|:-----|:-----|
| **短期（≤3 个月）** | 市场定价效率 | 市场指标（利差/ONRRP/TGA/信贷脉冲） |
| **中期（3 个月 ~ 2 年）** | 景气循环 + 政策博弈 | 景气指标（PMI）+ 政策 + 通胀/就业辅助 |
| **长期（>2 年）** | 结构性变迁 | 制度/结构指标 |

**如果用户没有明确时间周期**，默认同时覆盖短期（当前定位）和中期（趋势判断），短期权重 40%，中期权重 60%。

## 步骤 1：短期分析（≤ 3 个月）— 市场定价指标占主导

> **核心逻辑：3 个月内，最可靠的信号来自被市场效率定价的指标。**
> 中期的景气/通胀/就业分析在短期维度上精度不够。

### 1.1 资金面松紧

- **短期国债利率与政策利率（FFR）的差值** — 3 个月期国债收益率 vs 联邦基金利率，利差收窄 = 资金偏紧
- **国债拍卖结果** — 得标利率 vs 预发行利率（when-issued yield），尾部（tail）扩大 = 需求不足 = 资金偏紧。金十 `list_calendar` 中搜索"国债竞拍-得标利率"可获取拍卖数据

### 1.2 美联储流动性

- **美联储资产负债表变动** — 扩表 = 流动性宽松，缩表（QT）= 流动性收紧
- **ONRRP 余额**（隔夜逆回购）— 余额下降 = 市场上多余的套利资金在减少，流动性边际收紧；余额见底 ≈ 流动性从"过剩"进入"充足"区间
- **TGA 账户**（财政部一般账户）— TGA 余额大幅波动影响短期资金面：TGA 减少 = 财政存款释放到市场 = 流动性宽松

### 1.3 信贷与杠杆

- **M1/M2 剪刀差** — M1 增速 > M2 增速 = 资金活化 = 风险偏好上升
- **银行储备金余额** — 持续下降 = 准备金稀缺 = 可能触发 SOFR 异动
- **信贷脉冲** — 新增信贷/GDP 的边际变化，领先 PMI 约 2-3 个月

### 1.4 市场情绪快照

- **股市** — SPX 位置 + VIX，反映资金杠杆率和风险偏好
- **汇率** — DXY 方向反映国际资金流向
- **突发事件** — 地缘政治危机、政策突然转向，三日内就能改变短期宏观环境

### 短期输出格式

```
## 短期宏观定位（≤3M）
### 资金面
[利差状态 + 国债拍卖信号]
### 美联储流动性
[资产负债表 + ONRRP + TGA → 宽松/中性/偏紧]
### 信贷与杠杆
[M1/M2剪刀差 + 信贷脉冲 → 资金活化程度]
### 短期综合
[流动性环境] → [风险资产短期方向]
```

## 步骤 2：中期分析（3 个月 ~ 2 年）— 景气循环 + 政策占主导

> **核心逻辑：中期宏观的核心是判断景气循环的位置。PMI 是判断景气的首要指标，通胀和就业作为辅助验证。**
>
> ⚠️ **关键权重规则**：
> - 你想知道**"经济在什么位置"** → **PMI > 通胀/就业**（PMI 更直观，影响因素更少）
> - 你想知道**"美联储会怎么做"** → **通胀/就业 > PMI**（美联储双 mandate 直接锚定这两项）
> - **GDP 是滞后指标**，对中期景气判断帮助有限，仅作为交叉验证

### 2.1 PMI 子项拆解（景气判断的首要工具）

PMI 由多个子指标组成。**不只看总量，必须拆解子项之间的相对变动。**

| ISM 子指标 | 反映什么 | 上升意味着 |
|:-----|:-----|:-----|
| 业务活动 (Business Activity) | 当前产出 | 需求旺盛，生产活跃 |
| 新订单 (New Orders) | 未来需求 | 领先指标——订单先于生产 |
| 就业 (Employment) | 企业扩招意愿 | 企业对未来的信心 |
| 价格 (Prices Paid) | 成本端通胀压力 | 上游价格传导 |
| 库存 (Inventories) | 库存水位 | 上升 = 主动备货 or 被动积压 |
| 订单积压 (Order Backlog) | 产能利用率 | 上升 = 产能不足，下降 = 需求降温 |
| 供应商交付 (Supplier Deliveries) | 供应链压力 | 上升 = 交付变慢 = 需求 > 供给 |

**PMI 分析示例**（2026 年 5 月 ISM 服务业 PMI）：

> ISM 服务业 PMI：53.6 → 54.5，高于 53.8 预期
> - 业务活动 55.9 → 57.7 ✅
> - 新订单 53.5 → 57.3 ✅
> - 库存 53.1 → 62.5 ⚠️ 大幅跳增
> - 就业 48 → 47.9 ❌ 收缩
> - 价格 70.7 → 71.3 ⚠️ 高位
> - 订单积压 53 → 51.3 📉
> - 供应商交付 56.8 → 55.2 📉

解读：
1. **当前仍处于供不应求状态**：业务活动转好 + 新订单增加 + 价格上涨 → 景气处于相对高峰
2. **但即将进入"被动加库存"阶段**：库存大幅增加（+9.4）的同时订单积压在减少（-1.7），意味着当前产能在未来存在供给过剩风险。就业收缩和供应商交付降温也印证了"企业不愿过分扩张"
3. **整体结论**：服务业仍然景气，但景气循环位置上可能已经过峰或即将过峰

### 2.2 库存循环定位

库存循环由两个变量定义：**库存方向 × 通胀方向**。必须配合 PMI 子项做判断，不能单看库存总量。

```
库存增加 + 通胀降低 = 被动加库存 → 衰退早期（卖不掉，被动积压）
库存增加 + 通胀增加 = 主动加库存 → 扩张期（预期好，主动备货）
库存减少 + 通胀增加 = 被动减库存 → 过热期（不够卖，库存被消耗）
库存减少 + 通胀减少 = 主动减库存 → 衰退深化（主动去化）
```

| 阶段 | 库存 | 通胀 | 经济含义 | 典型政策 |
|:-----|:--:|:--:|:-----|:-----|
| 主动加库存 | ↑ | ↑ | 扩张期，企业对未来乐观 | 紧缩 |
| 被动减库存 | ↓ | ↑ | 过热期，供给跟不上需求 | 强力紧缩 |
| 被动加库存 | ↑ | ↓ | 放缓/衰退早期，需求退潮 | 宽松 |
| 主动减库存 | ↓ | ↓ | 衰退深化，企业主动收缩 | 强力宽松 |

⚠️ **库存数据限制**：全社会库存很难系统量化，现有数据无法覆盖所有行业。必须配合 PMI 库存分项 + 订单积压分项做判断。

### 2.3 CPI-PPI 剪刀差（利润率传导）

> 通胀不只是"高还是低"——**CPI 和 PPI 的差值决定了企业的利润空间，尤其是制造业。**

**使用前必须先判断通胀来源**：是景气驱动还是突发事件（疫情/战争）驱动？突发事件驱动的通胀不能用传统框架——它不能通过景气降温来解决。

**四种情景：**

| CPI | PPI | CPI-PPI 关系 | 利润率方向 | 景气含义 |
|:--:|:--:|:-----|:--:|:-----|
| 高 | 高 | CPI > PPI | 扩张 ✅ | 景气增长中期/加速期——企业将成本压力传导给消费端 |
| 高 | 高 | CPI < PPI | 压缩 ⚠️ | **最关键转折信号**——消费端不愿为高价买单，由好转坏或由坏转好的拐点 |
| 低 | 低 | CPI > PPI | 温和改善 🟢 | 复苏早期——成本端压力小，消费端仍在支付 |
| 低 | 低 | CPI < PPI | 极度压缩 🔴 | 衰退期——消费意愿低迷，售价甚至低于低位成本 |

### 2.4 就业市场结构分析

就业市场从四个维度拆解，**不只看非农总量**：

| 维度 | 指标 | 反映什么 |
|:-----|:-----|:-----|
| 需求端 | JOLTS 职位空缺 | 企业在招多少人 |
| 供给端 | 劳动参与率 | 有多少人愿意工作 |
| 匹配效率 | 非农新增就业 + 失业率 | 供需匹配得好不好 |
| 定价 | 平均时薪（环比） | 劳动力市场的均衡价格 |

**联合判断规则**：

```
需求↑ + 供给↑ + 匹配好 + 工资↑ = 景气增长中期/加速期 ✅
需求↑ + 供给↓ + 匹配差 + 工资↑ = 劳动力市场紧俏 → 工资通胀压力 ⚠️
需求↓ + 供给↑ + 匹配差 + 工资↓ = 景气减速 🟡
需求↓ + 供给↓ + 匹配差 + 工资↓ = 劳动力市场收缩 → 衰退信号 🔴
```

### 2.5 货币政策 + 财政政策

- **货币周期** — 主流央行（美联储为首）的利率周期，2 年内主导流动性松紧
- **财政政策** — 一届政府从上台到政策稳定约需 2 年，政策效力在此期间充分释放
- **政策与景气匹配度**：

| 景气状态 | 匹配政策 | 不匹配信号 |
|:-----|:-----|:-----|
| 过热 | 紧缩 | 还在宽松 → 通胀风险上升 |
| 放缓 | 宽松 | 还在紧缩 → 衰退风险上升 |
| 滞胀 | 两难 | 无论松紧都有副作用 |

### 2.6 指标权重分配规则（元规则）

> 不同问题用不同权重。这是整个中期分析框架的核心。

| 你想回答的问题 | PMI | 通胀 | 就业 | GDP | 市场指标 |
|:-----|:--:|:--:|:--:|:--:|:--:|
| 经济处于什么景气位置？ | 🔴 最高 | 🟡 辅助 | 🟡 辅助 | 🟢 滞后 | 🟢 减少 |
| 美联储下一步会做什么？ | 🟡 参考 | 🔴 最高 | 🔴 最高 | 🟡 参考 | 🟡 参考 |
| 市场未来 6 个月走势？ | 🔴 最高 | 🟡 辅助 | 🟡 辅助 | 🟢 滞后 | 🟡 参考 |
| 通胀什么时候见顶/底？ | 🟡 参考 | 🔴 最高 | 🟡 辅助 | 🟢 无关 | 🟡 参考 |

**如果 PMI 与通胀/就业出现矛盾**：PMI 说扩张但就业很差 → 对于"经济位置"，优先信 PMI。对于"美联储政策"，通胀/就业权重更高。

## 步骤 3：长期分析（> 2 年）— 结构性因素占主导

> **核心逻辑：长期宏观跳脱了常规框架，更接近于对社会的综合理解。**
> 市场没有可以直接参照的指标——如果有，也是市场定价的"结果"（如长天期国债利率方向），而非分析的"输入"。

### 可观察的结构性指标

1. **执政官员的政策倾向** — 美联储主席、财政部长、总统等的政策偏好和学术背景是预测长期政策方向的线索
2. **长期产业政策方向** — 倾向服务业还是制造业？技术自主还是全球分工？
3. **汇率变动反映的国际经贸格局** — 如日本从顺差转为逆差、货币持续贬值 → 一国在全球价值链中的地位变化
4. **综合国力的改变** — 财政状况（债务/GDP趋势）、地缘政治环境、人口结构（老龄化/劳动力供给）、科技发展水平

## 步骤 4：非常态分析（地缘 → 经贸 → 市场）

> **核心逻辑：地缘政治不直接影响市场，而是通过国际经贸环境传导。**
> 此步骤独立于时间周期——地缘事件可能影响任何周期。

传导链：
```
地缘事件 → 能源/供应链冲击 → 通胀/成本上升 → 货币政策预期 → 风险资产定价
```

1. 识别当前地缘热点事件
2. 判断影响路径：
   - **油价传导**（中东冲突 → 原油 → 通胀）
   - **供应链传导**（关税 → 成本 → 企业利润）
   - **避险传导**（战争 → 黄金/美债 → 风险偏好）
   - **政策转向**（制裁 → 各国央行调整储备结构）
3. 评估持续性和强度（短期脉冲 or 长期结构性）
4. 标注：该地缘事件是否已被市场定价

## 步骤 5：黑箱观察（外围市场价格联动）

> **核心逻辑：不分析过程，只看各市场结果之间的相互联动。**
> 黑箱模式独立于时间周期——它是对"当下市场在交易什么"的快照。

1. 收集各市场近期表现（方向 + 幅度）：美债收益率(2Y/10Y)、美股(SPX)、VIX、黄金、原油、DXY
2. 判断联动模式：

| 模式 | 信号 | 对应时间周期 |
|:-----|:-----|:-----|
| 股债齐跌 | 流动性危机/通胀恐慌 | 短期：资金面急剧收紧 |
| 股跌债涨 | 避险模式（衰退预期） | 中期：市场在定价衰退 |
| 黄金涨 + 美元涨 | 地缘避险 | 短期/中期 |
| 黄金涨 + 美元跌 | 美元信用担忧 | 长期：结构性变化 |
| VIX 飙升 + 风险资产跌 | 恐慌情绪 | 短期：流动性冲击 |
| 黄金涨 + 美股涨 | 非纯粹避险（流动性推动/通胀预期） | 中期：软着陆预期 |

3. 注意**背离信号**（如黄金涨但美股也在涨 → 不是纯粹避险）

## 注意事项（Pitfalls）

### 框架层面
- ⚠️ **先确定时间周期，再开始分析** — 不同周期用不同指标和权重
- ⚠️ **PMI 是景气判断的首要指标** — 不要被通胀或就业数据的噪音带偏
- ⚠️ **通胀是景气的结果，不是原因** — 必须判断通胀来源后再用剪刀差框架
- ⚠️ **CPI-PPI 剪刀差使用前，先判断通胀是景气驱动还是突发事件驱动**
- ⚠️ **GDP 是滞后指标** — 对中期景气判断帮助有限，仅作为交叉验证
- ⚠️ **指标权重取决于你要回答什么问题** — 参考步骤 2.6 的权重分配表

### 数据层面
- ❌ 不要跳因果 — 从「油价涨」到「BTC要跌」必须有完整传导链
- ❌ 不要用记忆中的数据报数字 — 所有数据来自工具返回，标注来源时间
- ❌ 不要把相关当因果 — 黄金涨和 BTC 涨同时发生不代表因果关系
- ❌ 金十 MCP 不可靠时不要反复重试 — 走降级链
- ✅ 金十 `list_calendar` 是美债拍卖数据的意外来源
- ✅ 金十 `list_flash` 对地缘事件捕获能力强 — 拉最近 20 条
- ✅ fin-skill `get_market_news` 是政策新闻 MVP — topic 可选 `fed`/`macro`/`china`

### 输出层面
- ✅ 预测性结论标注置信度：🟢高 🟡中 🔴低
- ✅ 在综合输出中说明本次的权重分配
- ✅ 区分"已被市场定价"和"未被市场定价"
