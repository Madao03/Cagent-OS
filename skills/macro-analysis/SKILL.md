# Macro Analysis Skill

> 宏观分析框架。对齐三层分析模式方法论（常态/非常态/黑箱），与 us-stock/crypto-analysis-skill 同结构。

## 何时触发

当用户询问或需要判断以下内容时，自动加载此 Skill：
- 宏观环境当前处于什么阶段（扩张/放缓/滞胀/衰退）
- 地缘政治事件对市场的影响路径
- 外围市场价格联动信号
- 宏观因素对 crypto 或风险资产的间接影响分析
- 美联储人事变动（主席换届、FOMC票委轮换、候选人立场分析）
- 央行政策拐点判断（降息/加息周期、QT/QE转向）
- 用户同时提出多个相互关联的宏观/市场问题（如「通胀+利率+科技股+IPO」之间的共振）

## 步骤 1：数据收集（能并行就并行）

> 所有数据务必标注来源和时间。不使用 LLM 记忆中的数字。
> 遇到数据源不可用，参考上方「数据源降级链」切换，不要阻塞。

尽可能同时获取以下四组数据：

1. **宏观景气数据** — 最新非农、CPI/PCE、PMI/ISM、GDP增速
   - 金十 MCP `list_calendar`（含 actual/consensus/previous）
   - 金十 MCP `list_flash`（实时快讯流，含经济数据公布瞬间的快讯）

2. **政策数据** — 当前利率水平、QT进展、近期关键政策/关税变动
   - fin-skill MCP `get_market_news(topic="fed")` — 对政策新闻发现能力极强
   - WebPlugin 搜索最新政策新闻 / 关税动态

3. **外围市场价格** — 美债收益率、美股、VIX、黄金、原油、DXY 的近期走势
   - 金十 MCP `get_quote(code="SPX")` — 首选
   - fin-skill MCP `get_stock_quote(symbol="SPY")` — 备选

4. **地缘政治事件** — 当前热点事件、最新进展
   - 金十 MCP `list_flash` — 对地缘事件捕获能力强，拉最近 20 条

---

## 步骤 2：常态性分析（宏观景气 ↔ 经济政策）

> 核心逻辑：景气与政策是双向对应关系，不同景气状态对应不同政策。

1. 判断当前景气所处阶段：
   - 扩张期（高增长+温和通胀）→ 政策倾向收紧
   - 放缓期（增长减速+通胀回落）→ 政策倾向宽松
   - 滞胀期（增长停滞+通胀高企）→ 政策两难
   - 衰退期（负增长+通缩）→ 强力宽松

2. 判断当前政策是滞后于景气、领先于景气、还是匹配：

   | 景气状态 | 匹配政策 | 不匹配信号 |
   |---------|---------|-----------|
   | 过热 | 紧缩 | 还在宽松 → 通胀风险上升 |
   | 放缓 | 宽松 | 还在紧缩 → 衰退风险上升 |
   | 滞胀 | 两难 | 无论松紧都有副作用 |

3. 输出结论：景气位置 + 政策匹配度

---

## 步骤 3：非常态性分析（地缘 → 经贸 → 市场）

> 核心逻辑：地缘政治不直接影响市场，而是通过国际经贸环境传导。

传导链：
```
地缘事件 → 能源/供应链冲击 → 通胀/成本上升 → 货币政策预期 → 风险资产定价
```

1. 识别当前地缘热点事件
2. 判断其对经贸的具体影响路径：
   - 油价传导（如中东冲突→原油→通胀）
   - 供应链传导（如关税→成本→企业利润）
   - 避险传导（如战争→黄金/美债→风险偏好）
3. 评估影响的持续性和强度（短期脉冲 or 长期结构性）
4. 输出结论：风险传导路径 + 影响程度

---

## 步骤 4：黑箱观察（外围市场价格联动）

> 核心逻辑：不分析过程，只看结果之间的相互联动。

1. 收集各市场近期表现（方向+幅度）：
   - 美债收益率曲线（2Y/10Y 利差变化）
   - 美股方向（SPX）
   - 恐慌指数（VIX 水平）
   - 避险资产（黄金方向）
   - 商品（原油方向）
   - 美元（DXY 方向）

2. 判断联动模式：

   | 模式 | 信号 |
   |-----|------|
   | 股债齐跌 | 流动性危机/通胀恐慌 |
   | 股跌债涨 | 避险模式（衰退预期）|
   | 黄金涨+美元涨 | 地缘避险 |
   | 黄金涨+美元跌 | 美元信用担忧 |
   | VIX 飙升+风险资产跌 | 恐慌情绪 |

3. 注意是否有背离信号（如黄金涨但美股也在涨 → 说明不是纯粹避险）

## 步骤 5：综合输出

将以上三层分析整合为一份宏观环境评估：

```markdown
## 宏观环境评估

### 景气周期定位
[当前所处阶段 + 证据]

### 政策评估
[政策匹配度 + 核心关注]

### 地缘风险传导
[风险点 + 传导路径 + 影响程度]

### 外围市场信号
[当前联动模式 + 关键背离/信号]

### 对风险资产的影响推演
[综合判断：利好/中性/利空 + 推演逻辑]

### 核心关注点
[未来1-4周最值得追踪的2-3个变量]
```

## 注意事项（Pitfalls）

- 不要跳因果 — 不能从「油价涨」直接跳到「BTC要跌」。必须有完整的传导链逻辑
- 不要用记忆中的数据报数字 — 所有数据必须来自工具返回，标注来源时间
- 不要把相关当因果 — 黄金涨和BTC涨同时发生不代表因果关系，优先用黑箱模式表述
- 景气↔政策是双向的 — 政策也会影响景气走向，不是单纯的政策响应
- 非常态分析要有传导链 — 必须写明：地缘→什么经贸变量→什么市场变量
- 标注置信度 — 预测性结论标注置信度：高 / 中 / 低
- 金十 MCP 不可靠时不要反复重试 — 走降级链，不要触发 tool loop
- 金十 `list_flash` 对地缘事件捕获能力强 — 地缘分析优先用 list_flash 拉最近 20 条而非搜索
- fin-skill `get_market_news` 是发现宏观新闻的 MVP — topic 可选 `fed` / `macro` / `china`
- 金十 `list_calendar` 是美债拍卖数据的意外来源 — 日历中含国债竞拍-得标利率条目
